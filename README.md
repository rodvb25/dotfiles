# Neovim-config

My neovim configuration with support for:

* Autocompletion
* LSP
* Treesitter
* Status Line
* Fuzzy Finder
* Git integration

# Installation

```bash
git clone "https://github.com/rodvb25/nvim-config.git"
cd nvim-config
ln -s ${PWD}/nvim/ ~/.config
```
This will not work if you already have a `./config/nvim` folder
